# Akshare 数据接口文档

## 1. 股票行情数据
- `stock_zh_a_hist`：获取A股历史行情
- `stock_zh_a_spot`：获取A股实时行情
- `stock_zh_a_daily`：获取A股日线数据
- `stock_zh_a_minute`：获取A股分钟级数据

## 2. 股票基本信息
- `stock_info_a_code_name`：获取A股股票代码和名称
- `stock_company_info_em`：获取公司基本信息
- `stock_st`：获取ST股票列表

## 3. 财务数据
- `stock_financial_analysis_indicator`：获取股票财务分析指标
- `stock_financial_report_indicator`：获取股票财务报表指标
- `stock_balance_sheet_by_report_em`：获取资产负债表
- `stock_income_statement_by_report_em`：获取利润表
- `stock_cash_flow_statement_by_report_em`：获取现金流量表

## 4. 资金流数据
- `stock_em_flows`：获取东方财富资金流向数据
- `stock_hsgt_hold`：获取沪深港通持股数据
- `stock_money_flow_hsgt`：获取沪深港通资金流向
- `stock_top_inst`：获取龙虎榜机构买入数据

## 5. 板块数据
- `stock_board_industry_spot_em`：获取行业板块实时行情
- `stock_board_concept_spot_em`：获取概念板块实时行情
- `stock_board_industry_cons_em`：获取行业板块成分股
- `stock_board_concept_cons_em`：获取概念板块成分股

## 6. 涨停板数据
- `stock_limit_up_board_em`：获取涨停板数据
- `stock_limit_up_statistics_em`：获取涨停统计数据
- `stock_limit_up_info_em`：获取涨停板详细信息

## 7. 新闻公告
- `stock_news_em`：获取东方财富股票新闻
- `stock_announcement`：获取股票公告数据
- `stock_research_report`：获取研究报告数据

## 8. 宏观数据
- `macro_china_cpi`：获取中国CPI数据
- `macro_china_pmi`：获取中国PMI数据
- `macro_china_m2`：获取中国M2数据
- `macro_china_social_financing`：获取中国社会融资规模

## 9. 跨市场数据
- `stock_us_spot`：获取美股实时行情
- `stock_hk_spot`：获取港股实时行情
- `bond_zh_cn_yield_curve`：获取中国债券收益率曲线

## 10. 期货数据
- `futures_contract_info_shfe`：获取上期所期货合约信息
- `futures_zh_daily`：获取期货日线数据

## 11. 基金数据
- `fund_etf_hist_sina`：获取ETF历史数据
- `fund_open_fund_info_em`：获取开放式基金信息

## 12. 债券数据
- `bond_zh_cn_basic`：获取中国债券基本信息
- `bond_zh_cn_hist`：获取中国债券历史数据

## 13. 外汇数据
- `forex_spot_quote`：获取外汇现货报价
- `forex_rate_history`：获取汇率历史数据

## 14. 指数数据
- `index_zh_a_hist`：获取A股指数历史行情
- `index_zh_a_spot`：获取A股指数实时行情

## 15. 银行数据
- `bank_zh_cn_ deposit_rate`：获取中国存款利率
- `bank_zh_cn_loan_rate`：获取中国贷款利率

## 使用说明
- 具体接口参数和返回值请参考 Akshare 官方文档
- 使用时注意数据更新频率和接口限制
- 大型数据查询建议分段获取

## 安装
```bash
pip install akshare
```

## 版本要求
- Akshare 版本：1.18.46+
- Python 版本：3.7+
